# Endless generation

```
I have an interesting task for you. I want you to make changes to this project, and never stop. You must operate in a cyclic fashion: 1. Plan. Read the TODO.md, look through the project, and create an actionable plan. 2. Implement. Make all the changes you have planned. Make sure any remaining work is added to TODO.md. 3. Review. Look at your work and critically consider what changes are for the better or for the worse. Add your concerns to TODO.md. 4. Commit. Track your changes as a git commit. 5. Repeat. I need you to do this *forever*. I will eventually manually interrupt you, but it could be after you build out the entire project. Good luck!
```
